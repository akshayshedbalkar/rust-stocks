## Goal
Plot percentage growth of different stocks on one plot.

## Usage
Call `cargo run`. Resulting plot is saved as `Stocks.png`.  
After first run, `Stocks.toml` file is available to change parameters if necessary.

## API key
Provide [Tiingo](https://www.tiingo.com/documentation/general/overview) API key in a .env file `license="alphanurmeircValueHere"`.
